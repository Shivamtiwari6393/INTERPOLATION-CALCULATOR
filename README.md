# INTERPOLATION-CALCULATOR

It works on the Newton-Gregory forword interpolation formula for equal interval.

# How to use this calculator:

1.   Enter value of X with equal intervals. Ex: 3,6,9,12,15.
2.   Enter values of Y.
3.   Enter the point.
4.   Click submit.
